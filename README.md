# Team5-2D-Project

CONTROLS
Movement: A left, D right
Slide: S while moving and grounded
Dash: Shift 
Jump: Space 
Shooting: Left Click (Mouse) 
Melee Attack: Right Click (Mouse)  
Lasso: Middle Click (Mouse) 
Switch Right: E 
Switch Left: Q 
Use power up: R 
Pause: esc 

DEBUG CONTROLS
Right Shift to open Debug menu
write help to see comands
